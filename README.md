# retrieval-kit

Typescript utilities for building knowledge retrieval pipelines.

## Scripts

| Command            |                                          |
| ------------------ | ---------------------------------------- |
| `pnpm build`       | Build all packages                       |
| `pnpm dev`         | Watch mode                               |
| `pnpm lint`        | Lint                                     |
| `pnpm typecheck`   | Type-check                               |
| `pnpm format`      | Prettier across the repo                 |
| `pnpm new-package` | Scaffold a new package from the template |
